# Workflow Automation

## What to Do

Build simple automated workflows using tools like Zapier, Make, or built-in software features so routine processes happen without your input. Start with your highest-impact automation candidate from the Task Audit (Action 1) and get it running this week.

## Why It Matters

Every manual step in a recurring workflow is a step that depends on someone remembering to do it. Automated workflows run every time, on time, without reminders. A lead fills in a form and automatically gets added to your CRM, receives a welcome email, and creates a task for you to follow up - all without you touching anything. This is not futuristic technology. It is available now for £20-50/month and most automations take under an hour to set up.

## How to Use This

Pick one automation, build it, test it, and get it running. Then add the next one. One at a time.

---

## Choose Your First Automation

- [ ] Review your top 3 candidates from the Task Audit (Action 1)
- [ ] ⚡ Quick Win: Start with the simplest one that saves the most time. Common first automations:
  - New form submission > auto-email response + add to CRM
  - New enquiry > Slack/email notification to you
  - Invoice overdue > automatic reminder email to client
  - New client signed > trigger onboarding email sequence
  - Social media post scheduled from a content calendar

## Build It

- [ ] Choose your automation tool:
  - **Zapier** (beginner-friendly, widest range of app connections, free tier available)
  - **Make (formerly Integromat)** (more powerful, slightly steeper learning curve, better value)
  - **Built-in features** (many tools like HubSpot, Mailchimp, Xero have their own automation)
- [ ] Map the workflow: Trigger (what starts it) > Actions (what happens automatically)
- [ ] Set it up step by step:
  - Define the trigger: "When [event] happens in [app]..."
  - Define the action(s): "Do [this] in [app]..."
  - Map the data: which fields from the trigger populate which fields in the action?
- [ ] Test with a real example: trigger the automation and check that every step works correctly
- [ ] Fix any issues and test again until it runs cleanly

## Common Automation Recipes

- [ ] **Lead capture:** Website form > CRM contact created > Welcome email sent > Task created for follow-up
- [ ] **Invoice chasing:** Invoice overdue by 7 days > Reminder email sent > Overdue by 14 days > Second reminder + notification to you
- [ ] **Client onboarding:** Contract signed > Welcome email with next steps > Calendar invite for kickoff > Folder created in shared drive
- [ ] **Social scheduling:** Content calendar updated > Post automatically published at scheduled time
- [ ] **Report generation:** End of week > Pull data from sources > Format into report > Email to you or team

## Monitor and Expand

- [ ] Check your first automation weekly for the first month: is it running correctly? Any errors?
- [ ] Calculate time saved: how many minutes per week has this automation freed up?
- [ ] Once stable, build your second automation from the priority list
- [ ] Aim for 3-5 working automations within 90 days
- [ ] Set a quarterly review: are automations still running? Do any need updating? What should be automated next?

## Expected Outcome

- At least one workflow fully automated and running reliably
- Time reclaimed from manual, repetitive work
- Fewer dropped balls on routine tasks (follow-ups, reminders, notifications)
- A growing library of automations that compound time savings
- Confidence that "automation" is not scary or complex - it is practical and accessible

## Coach Notes

The biggest blocker is overwhelm: clients see automation possibilities everywhere and freeze. Force a single starting point: "What one manual process annoys you most?" Build that one, celebrate the win, then move to the next. Zapier is the easiest entry point for non-technical users - their visual workflow builder is intuitive. Warn clients about over-automating: not every process should be automated. Automate the repetitive and rule-based; keep the human touch where relationships matter. A personalised follow-up email from you is worth more than an automated one, even if it takes longer.
