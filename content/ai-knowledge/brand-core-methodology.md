# BOSS core methodology (compact)

**BOSS** (Business Owner Strategy System): diagnose → sequence → track.

- **Diagnostic:** 50 questions across 10 areas + Owner Performance; scores 0/1/2; max 100. **BOSS Wheel** sells; **BOSS Grid** solves.
- **Pillars:** Control (visibility, structure), Velocity (cash, quick wins), Value (growth, people, strategy).
- **ROI proof:** Funnel multiplier (small % lifts compound); time recapture (£10/£100/£1k tasks); margin insight.
- **BCA vs BOSS:** BCA trains coaches in the Profit System; BOSS implements the same framework in owners’ businesses. Same skeleton, different buyer (coach vs owner).
